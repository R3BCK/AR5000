=====
Режимы сканирования и поиска
- [ ]  SM / MS — запуск сканирования памяти
- [ ]  SS / VS — запуск поиска по диапазону
- [ ]  SE / SR — настройка 20 банков поиска
- [ ]  PS / PD / PR — управление списком пропускаемых частот (PASS List, 2100 записей)
- [ ]  AS — автозапись найденных частот
- [ ]  QS — очистка банков поиска

Расширенные настройки
- [ ]  BM / ML / GM — группировка банков сканирования
- [ ]  GA / GR — выборочное сканирование (Select Scan)
- [ ]  XD / XP / XM — задержка, пауза, режимы сканирования
- [ ]  SA / SB / XA / XB — голосовой и пороговой скан
- [ ]  SD / SP — время удержания/паузы в поиске


Специальные функции
- [ ]  AI — управление выходом EXT-IF (для SDU5000)
- [ ]  AN — выбор антенны по диапазону (Auto/1/2/3/4)
- [ ]  EN / LP / HP — де-эмфазис, ФВЧ, ФНЧ (ручной режим)
- [ ]  NB / AF / QM / LS — шумоподавитель, AFC, DTMF, Tone Eliminate
- [ ]  MT / TU — ручная/автоматическая подстройка преселектора
- [ ]  DS — Cyberscan (только +3 версия)
- [ ]  SC — скремблер данных
- [ ]  CN — CTCSS

Системные команды
- [ ]  EX — выход из remote-режима
- [ ]  QP / X — выключение/включение питания
- [ ]  VR — чтение версии прошивки
- [ ]  LC / LM — возврат данных при открытии сквоша / показания S-метра
- [ ]  RX — дамп всех текущих параметров
